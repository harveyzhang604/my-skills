# 山音超级导演大师｜Claude Director Agent
> 纯自然语言 Claude Skills 文件 | 影视导演智能Agent | 分镜表自动生成
> 联动：[山音超级编剧大师](https://github.com/Shanyin-ai/shanyin-screenwriting-master)

## 🎬 项目介绍
**山音超级导演大师**是为 Claude 设计的**纯自然语言导演技能**，无任何编程语言，开箱即用。
核心能力：从剧本一键生成**工业标准九列分镜表（xlsx）**，内置顶级导演视听理论+全类型风格模板，支持全格式影片分镜拆解。

## ✨ 核心功能
- ✅ 导演定调 / 节奏规划 / 剧本微调 / 分镜全拆解
- ✅ 数十种类型风格模板，支持交叉组合
- ✅ 全局强触发，极简指令即可调用
- ✅ 联动「山音超级编剧大师」形成**编剧→导演**全流程
- ✅ 输出标准九列 xlsx 分镜表，可直接用于拍摄

## 🚀 使用方法
1. 复制本仓库 `claude-skill-shanyin-super-director.md` 全部内容
2. 粘贴至 Claude 自定义技能/Skills 配置（或直接对话中粘贴激活）
3. 输入触发指令（如「帮我拆分镜」「剧本转分镜表」）即可启动

## 🎯 触发关键词（全覆盖）
分镜、分镜表、导演、镜头语言、运镜、构图、景别、剪辑节奏、视听风格、摄影角度、导演风格、拍摄方案、分镜拆解、storyboard、这场戏怎么拍、给我一个导演方案
